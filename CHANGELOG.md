# Changelog

All notable changes to this research project are documented in this file.

This project follows a conservative change policy:
- Substantive analytical changes are explicitly noted
- Editorial or formatting changes are distinguished from content changes
- Post-publication updates are additive and transparent

---

## [1.0.0] — 2026-01-03

### Added
- Full research paper: *System Prompt Forensics: Governance Structures in AI Developer Assistants*
- Appendix defining **Prompt Governance Primitives (PGPs)**
- Executive Brief (2-page decision-maker summary)
- Board-Level Brief (1-page governance-focused summary)
- PGP taxonomy diagram (Mermaid)
- Repository README describing scope, methodology, and artifacts
- AI Assistance Disclosure (`DISCLOSURE.md`)
- Citation metadata (`CITATION.cff`)
- Reviewer clarification template (`reviewer-response.md`)

### Research Scope
- Comparative analysis of system prompts across multiple AI developer assistants and modes
- Formalization of recurring prompt-level governance patterns
- Introduction of Prompt Governance Primitives as reusable abstractions

### Disclosure
- Documented assistive use of GPT-5.2 for drafting, editorial refinement, synthesis, and diagram generation
- Explicit confirmation that all analysis, interpretations, and conclusions are author-derived and verified

### Notes
- This release constitutes the **initial public publication**
- Core findings and taxonomy are considered **stable**
- Future updates will be versioned and documented below
